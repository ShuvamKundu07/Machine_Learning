# API to DataFrame

This project demonstrates how to fetch movie data from the **IMDb236 API** on RapidAPI and convert the API response into a structured Pandas DataFrame for data analysis.

## What I Learned

* Set up and use the IMDb236 API through RapidAPI (https://rapidapi.com/hub).
* Securely store API credentials using a `.env` file and `python-dotenv`.
* Send HTTP GET requests using the `requests` library.
* Parse JSON responses returned by the API.
* Convert JSON data into a Pandas DataFrame.
* Select and inspect relevant columns from the dataset.
* Configure Pandas display options for better visualization in Jupyter Notebook.
* Export the processed data to a CSV file for further analysis.

## Technologies Used

* Python
* Pandas
* Requests
* Python-dotenv
* Jupyter Notebook
* RapidAPI (IMDb236 API)

## Output

The notebook fetches movie information from the IMDb236 API and saves the processed data as a CSV file, making it ready for data analysis or machine learning tasks.
